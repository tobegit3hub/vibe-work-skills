# 运营文案检查器

这是一个专为运营文案设计的综合检查工具，帮助确保文案质量达到发布标准。

## 检查流程

请按照以下六个步骤逐一检查文案：

---

## 步骤一：错别字检查

仔细检查文案中的文字错误，包括：

- **同音字错误**：如"在/再"、"的/地/得"、"做/作"混用
- **形近字错误**：如"已/己"、"未/末"混淆
- **多字/少字**：漏字或重复字
- **拼音输入错误**：输入法导致的常见错误
- **专业术语错误**：行业用语或品牌名称拼写错误

**输出格式**：
```
【错别字检查结果】
- 位置：[段落/行号] | 原文："xxx" → 修正："xxx" | 错误类型：[类型]
```

如无错别字，输出：`【错别字检查结果】未发现错别字。`

---

## 步骤二：内容清晰度检查

评估文案是否清晰易懂，重点检查：

- **核心信息明确**：主要卖点/活动内容是否一目了然
- **语句通顺**：是否存在病句、语序不当
- **逻辑连贯**：上下文衔接是否自然流畅
- **用词准确**：是否存在歧义或表述模糊
- **目标受众适配**：语言风格是否符合目标用户群体

**评分标准**（1-5分）：
- 5分：非常清晰，一读即懂
- 4分：较为清晰，基本流畅
- 3分：基本清晰，部分需优化
- 2分：较为混乱，需要改进
- 1分：很难理解，需重写

**输出格式**：
```
【内容清晰度评估】
- 清晰度评分：X/5分
- 核心信息：[提取的核心信息]
- 问题点：
  1. [问题描述] → 建议修改："xxx"
- 总体评价：[简要评价]
```

---

## 步骤三：图文并茂检查

评估文案的视觉呈现效果和图文配合程度：

- **配图建议**：文案是否需要配图，需要什么类型的图片
- **排版结构**：段落划分是否合理，是否易于阅读
- **视觉元素**：是否需要添加emoji、分隔符、项目符号等
- **重点突出**：关键信息是否有视觉强调（加粗、颜色等）
- **长度适配**：文案长度是否适合发布渠道

**输出格式**：
```
【图文并茂检查】
- 当前状态：[纯文字/有配图说明/有emoji等]
- 配图建议：[是/否需要配图，建议配什么类型的图]
- 排版建议：
  1. [具体建议]
- 视觉优化建议：
  1. [具体建议]
```

---

## 步骤四：营销效果检查

评估文案的营销吸引力：

- **标题/开头吸引力**：是否能抓住读者注意力
- **价值主张**：利益点是否突出、是否戳中用户痛点
- **行动号召（CTA）**：是否有明确的行动引导
- **紧迫感/稀缺性**：是否有促进转化的元素
- **信任背书**：是否有增强可信度的内容

**输出格式**：
```
【营销效果检查】
- 吸引力评分：X/5分
- 标题/开头评价：[评价]
- 价值主张：[是否清晰] - [具体内容]
- CTA分析：[有/无] - [具体内容或建议添加]
- 优化建议：
  1. [具体建议]
```

---

## 步骤五：合规性检查

检查文案是否符合平台规范和法律法规：

- **违禁词检查**：是否包含广告法违禁词（最、第一、绝对等极限词）
- **敏感词检查**：是否涉及敏感话题或不当内容
- **承诺合规**：是否有无法兑现的承诺或虚假宣传
- **版权风险**：是否涉及他人商标、版权内容
- **平台规范**：是否符合发布平台的内容规范

**输出格式**：
```
【合规性检查】
- 违禁词：[有/无] - [具体词汇及位置]
- 敏感内容：[有/无] - [说明]
- 风险点：
  1. [风险描述] → 建议修改："xxx"
- 合规评估：[通过/需修改]
```

---

## 步骤六：综合改进建议

基于以上所有检查，提供整体优化方案：

- **优先级排序**：按重要程度列出需要修改的问题
- **优化版本**：提供修改后的文案参考
- **A/B测试建议**：如有必要，提供可对比测试的版本

**输出格式**：
```
【综合改进建议】

一、问题优先级清单：
1. 【紧急】[问题]
2. 【重要】[问题]
3. 【建议】[问题]

二、优化后文案：
---
[修改后的完整文案]
---

三、优化说明：
- [修改点1及原因]
- [修改点2及原因]

四、整体评分：X/10分
```
